Agent Design

Coordinator & Worker Agents

Coordinator Agent

Orchestrates the full pipeline · Routes data between agents · Manages state · Enforces sequence & error handling

adk.py · AgentPipeline
dispatches
WORKER AGENT 01
🔭

Reconnaissance Agent

Performs port scans, banner grabbing, DNS record analysis, and SSL certificate inspection to map the full attack surface.

WORKER AGENT 02
🐛

Vulnerability Agent

Cross-references discovered services with known CVEs, misconfigurations, and severity classifications using Gemini.

WORKER AGENT 03
📊

WORKER AGENT 04
📝

Report Generation Agent

Synthesises all agent outputs into a structured executive Markdown report with CVSS summaries and remediation steps.

Core Technologies

ADK · MCP Server · Gemini

⚙️

Agent Development Kit (ADK)

Custom-built orchestration framework defining Agent, Task & Workflow abstractions for the multi-agent pipeline.

  • Modular agent definition & sequencing
  • Task context passing between agents
  • Error isolation per agent stage
  • Extendable workflow engine
  • adk.py · agents.py integration
🔌

MCP Server

FastMCP-powered server exposing SentinelAI scans directly to AI agentic workflows and MCP-compatible clients.

  • run_security_scan tool exposed
  • run_custom_security_scan tool
  • Compatible with Claude Desktop
  • Works with Gemini IDEs
  • Enables agent-to-agent calling
🤖

Gemini LLM

Google Generative AI (Gemini) powers all four agents — enabling natural language vulnerability analysis and report generation.

  • CVE reasoning & mapping
  • Business risk contextualisation
  • Executive report authoring
  • AES-256 encrypted API key storage
  • google-generativeai SDK

Security & Deployment

Built-in Security & Deployment

🛡️ Security Hardening

✔
Input Validation — Strict regex checks on domain syntax, IP formats, port lists to block injection attacks
✔
Rate Limiting — TokenBucketRateLimiter prevents API exhaustion and brute-force abuse
✔
AES-256-GCM Encryption — API keys never stored in plaintext; protected with PBKDF2HMAC key derivation
✔
Sandboxed Scanning — Network scanner isolated from backend logic to prevent privilege escalation
✔
Error Containment — Each agent stage isolated; one failure does not cascade the pipeline

🚀 Deployment Interfaces

🌐
Web Dashboard — Glassmorphic premium UI with real-time scan results, risk scores & visual metrics
💻
CLI Tool — backend/cli.py for terminal-first workflows; supports --profile and --output flags
🔌
MCP Server — Expose scans to Claude Desktop, Gemini IDEs, and any MCP-compatible agent workflow
🐳
Docker Deployment — docker-compose up --build; runs offline at localhost:8000 in minutes
☁️
Vercel Cloud Deploy — Live at sentinel-ai-kappa-teal.vercel.app — try without any local setup
